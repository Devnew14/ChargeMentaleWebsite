# Projet de Licence : Analyse de la Charge Mentale

## Contexte
Dans le cadre de notre deuxième année de licence en informatique, nous avons réalisé un projet sur la **charge mentale d’une personne**. L’objectif était de créer une **application mobile** capable de mesurer la charge mentale d’un individu ou d’un groupe, et d’exploiter ces données à travers un **site web** développé en parallèle.  

Le site permet d’analyser la charge mentale au sein d’un groupe de **pilotes dans le domaine de l’aviation**, afin d’étudier les performances d’un équipage durant un vol et les conséquences sur les tâches effectuées.  

L’ultime objectif de ce projet est de rendre les résultats **significatifs et exploitables**, pour éventuellement corriger les erreurs humaines dans le domaine de l’aviation civile.

---

## Fonctionnalités principales

### Application mobile
- Mesure de la charge mentale individuelle.
- Suivi des performances sur plusieurs vols ou missions.
- Interface simple et intuitive pour l’utilisateur.
- Développée en **JavaScript**.

### Site web
- Visualisation des résultats collectés depuis l’application mobile.
- Analyse de la charge mentale au niveau d’un groupe de pilotes.
- Gestion des utilisateurs avec **authentification Firebase**.
- Développement avec **HTML, CSS et JavaScript**.
- Suivi des performances et des tâches effectuées.

---

## Technologies utilisées
- **Front-end mobile** : JavaScript  
- **Site web** : HTML, CSS, JavaScript  
- **Backend et authentification** : Firebase (gestion des comptes et mots de passe)  
- **Outils de prototypage** : Maquettes et documents de conception pour structurer le projet
